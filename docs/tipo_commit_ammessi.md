Tipo Quando usarlo
feat Una nuova funzionalità
fix La correzione di un bug
chore Manutenzione (configurazioni, pacchetti, husky)
docs Modifiche solo alla documentazione
style Formattazione, punti e virgola mancanti (no modifiche al codice)
refactor Modifiche al codice che non correggono bug né aggiungono funzioni
perf Modifiche per migliorare le prestazioni
test Aggiunta o modifica di test
